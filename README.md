# Smart Attendance Calculator 🎓

A Python project to track student attendance, calculate percentages, predict future attendance, and find safe leave limits.

---

## 🚀 Features

- Student-wise attendance calculation
- Subject-wise percentage
- Overall attendance calculation
- Future attendance prediction
- Safe leave calculator
- CSV file export

---

## 🛠️ Tech Stack

- Python 3
- CSV Module

---

## 📸 Output Example

- Attendance percentage per subject
- Overall attendance
- Prediction system
- Safe leave calculator
