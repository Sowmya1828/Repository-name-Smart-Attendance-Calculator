# Smart Attendance Calculator 🎓

A Python-based console application to help students monitor their attendance, calculate subject-wise and overall attendance percentages, predict future attendance after planned absences, and determine the maximum number of classes they can safely miss while maintaining the minimum attendance requirement. The application also exports attendance data to a CSV file for future reference.

---

## 🚀 Features

- Student-wise attendance calculation
- Subject-wise attendance percentage
- Overall attendance calculation
- Future attendance prediction
- Safe Leave Calculator (75% attendance rule)
- Automatic CSV file export
- Simple and user-friendly command-line interface

---

## 🛠️ Tech Stack

- Python 3
- CSV Module
